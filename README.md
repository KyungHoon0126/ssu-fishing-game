## 🚀 Getting Started
1. 저장소 클론 또는 ZIP 다운로드
2. `fishing-game/index.html`을 브라우저로 열기  
   - VS Code 사용 시 **Live Server** 확장을 추천
3. 사운드가 재생되지 않을 경우 브라우저의 오토플레이 제한을 확인

## 🎮 Controls
- **MENU**: 시즌 탭 클릭 → `시작하기` 버튼
- **INFO 화면**: `ENTER` 게임 시작, `X` 닫기, 휠 스크롤로 설명 읽기
- **PLAY**  
  - 마우스 이동: 배 위치  
  - 마우스 클릭: 낚싯바늘 올리기/내리기  
  - `SPACE`: 게이지 타이밍  
- **RESULT**: `ENTER` 재시작
